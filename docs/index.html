Features

Everything in one window.

No browser tabs, no context switching. Terminal, editor, AI, and tools — unified.

⌘

Terminal

xterm.js with WebGL rendering. Multi-tab, split panes, shell integration for bash, zsh, PowerShell, and cmd.

✎

Code Editor

CodeMirror 6 with 44 languages, find & replace, inline AI autocomplete, Vim mode, and 9 themes.

◆

AI Agent

Reads, writes, and runs code. Edit diffs with approval flow, auto-approve modes, context summarization.

⚡

MCP Support

Connect external tool servers via Model Context Protocol. Browse and install from the official registry.

🎨

Image & Video Gen

Generate images (GPT Image 2, Nano Banana 2, Grok Imagine, ComfyUI) and videos (Kling 3.0, Veo 3.1, Seedance 2.0, ComfyUI) from chat. Inline preview with lightbox.

◫

File Explorer

Catppuccin icon theme, fuzzy search, inline rename, image & PDF preview, drag to attach to AI.

⊞

Web Preview & API Tester

Built-in browser for local dev servers. REST API tester with headers, body editor, and JSON pretty-print.
